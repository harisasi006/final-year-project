# 🎯 Placement Preparation Helper - Java Console Application

A Java-based console application to help final year students prepare for campus placements.

---

## 📌 Features

| Feature | Description |
|---|---|
| 📘 DSA Questions | Top 10 Data Structures & Algorithms Q&A with explanations |
| 🧠 Aptitude Quiz | 10 MCQ questions with score tracking and explanations |
| 🎤 HR Interview | Top 10 HR questions with sample answers |
| 📊 Score Tracker | View your aptitude quiz score and grade |

---

## 🗂️ Project Structure

```
PlacementHelper/
├── src/
│   ├── Main.java          → Entry point, main menu
│   ├── DSAQuestions.java  → DSA Q&A module
│   ├── AptitudeQuiz.java  → MCQ Quiz with scoring
│   ├── HRInterview.java   → HR Interview Q&A
│   └── ScoreTracker.java  → Score display module
└── README.md
```

## ▶️ How to Run

### Option 1: Online (No Installation)
1. Copy the code from any `.java` file
2. Paste it in [OnlineGDB](https://www.onlinegdb.com/online_java_compiler)
3. Run it!

### Option 2: Command Line
```bash
# Step 1: Go to src folder
cd PlacementHelper/src

# Step 2: Compile all files
javac *.java

# Step 3: Run the program
java Main
```

### Option 3: IDE (IntelliJ / Eclipse)
1. Open the `src` folder as a project
2. Run `Main.java`
